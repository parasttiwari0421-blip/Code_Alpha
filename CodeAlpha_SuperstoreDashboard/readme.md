# Superstore Sales & Profit Analysis Dashboard

## CodeAlpha Data Analytics Internship - Task 3

This project focuses on analyzing Superstore sales data using Python data visualization libraries.

The goal of this project is to identify:
- Sales trends
- Profit analysis
- Regional performance
- Category-wise performance
- Business insights

---

# Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

---

# Dataset Used

Sample Superstore Dataset

---

# Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Business Insights

---

# Visualizations Included

- Sales by Region
- Profit by Category
- Sales vs Profit Analysis
- Top 10 States by Sales
- Monthly Sales Trend
- Correlation Heatmap
- Category-wise Sales Distribution
- Sub-Category Wise Profit

---

# Key Insights

- West region generated the highest sales
- Technology category earned the highest profit
- Discounts negatively impacted profits
- California recorded the highest sales
- Some sub-categories generated losses

---

# Conclusion

This project helped in understanding business performance through visualization techniques and generated meaningful insights from sales data.
