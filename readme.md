This project performs Exploratory Data Analysis (EDA) on a global layoffs dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Tech companies had highest layoffs
- USA recorded maximum layoffs
- Layoffs increased during recession periods
